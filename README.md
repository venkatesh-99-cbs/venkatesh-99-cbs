## 👤 About Me

I'm a Cybersecurity-focused Computer Science student who learns best by building real systems and then trying to break them. My work sits at the intersection of **secure software engineering** and **offensive/defensive security** — I write the Django backend in the morning and run the Nmap scan against it in the evening.

- 🔭 Currently engineering **AI-assisted cybersecurity tooling** and **production full-stack web applications**
- 🛡️ Training in **SOC operations, threat detection, and incident response** as part of a structured cybersecurity program (Team: *Cyber Wizards*)
- 🧠 Believer in security-by-design — I treat authentication flows, payment webhooks, and file uploads as attack surfaces, not just features
- 🤝 Open to collaborating on **open-source security tooling, AI integrations, and full-stack builds**
- 💬 Ask me about **Python, Django, Linux internals, malware analysis basics, or full-stack architecture**
